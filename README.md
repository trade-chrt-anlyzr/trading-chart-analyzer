# Trading Chart Analyzer ‚Äî OCR + Price Action + AI Logic

This web-based application allows traders to upload a screenshot of a chart and receive:

- Automatic chart analysis  
- BUY/SELL entry suggestions  
- Stop Loss and Take Profit levels  
- Detection of OB, FVG, BOS, CHOCH, Liquidity Sweeps  
- Price mapping using OCR of the chart's price axis  
- Confidence scoring per signal  

This repo contains both:

- **Backend** (Flask + Python + OCR + logic)
- **Frontend** (React + Vite)

And supports:

- **Docker**  
- **Render.com deployment**  
- **Local development**
